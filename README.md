# Fresh Admin Web (GitHub Pages Ready)

This package contains only the Admin Web app and a GitHub Actions workflow to build & deploy to GitHub Pages.

## Quick start
1. Create a new GitHub repository.
2. Upload the contents of this zip (keep folders).
3. Go to **Settings → Pages → Source = GitHub Actions**.
4. Go to **Actions → build-and-deploy-web-only → Run workflow**.
5. Your Pages URL will appear under **Settings → Pages** after it finishes.
